THE FIRST PYTHON PROJECT

The first project, which was written on python language. This is the old skull game from 80-90 years. The game is console-text oriented. It has map wich show on rooms and ways, where user can go.

Main menu (Click for zoom)

On this screen, you can see the simple menu which made on standart while loop. Here you can start or continue game

After ther user has chosen New Game ( shortcut NG ),The screen will be changed to main game view (Map view ) Where the player can play.

Next screenshot will show this main game view

Main game view
Main game view

The player can move by typing the shortcut of allowed exits, it can be North (N), South (S), West (W), East (E). Some rooms have only exits and some rooms have quests, which user may complete and some on them are required for winning the game. Also the rooms contain items and tools, which will be useful for quests. Users also can use cheat codes such as teleport (TT+room number)
